# DealSmart

DealSmart is a full-stack application with separate frontend and backend components.

## Project Structure

```
DealSmart/
├── Backend/              # Backend API and server code
│   ├── DealSmartBackend/ # Main backend application
│   └── DealSmartdump.sql # Database dump file
└── Frontend/             # Frontend client application
    └── DealSmartFrontend/ # Main frontend application
```

## Getting Started

### Prerequisites

- Node.js (for frontend)
- Database server (MySQL/PostgreSQL)
- Backend runtime environment

### Setup

1. **Backend Setup**
   - Navigate to `Backend/DealSmartBackend/`
   - Follow the setup instructions in the Backend README

2. **Frontend Setup**
   - Navigate to `Frontend/DealSmartFrontend/`
   - Follow the setup instructions in the Frontend README

3. **Database Setup**
   - Import `Backend/DealSmartdump.sql` into your database
   - Configure database connection in backend settings

## Development

- Backend and Frontend can be developed and run independently
- Ensure the backend API is running before starting the frontend development server
